2-1-4. Logical Operators :

Python code: <%# Copy source code %> 
x = 5
print(x < 8 and x > 3) # True         like && in Javascript
print(x < 8 and x < 3) # False
print(x < 8 or x < 3) # True          like || in Javascript
print(not(x < 8 and x > 3)) # False   like !  in Javascript
Javascript code: 
var x = 5;       
document.write(x < 8 && x > 3); // true         like "and" in Python
document.write(x < 8 && x < 3); // false
document.write(x < 8 || x < 3); // true         like "or" in Python
document.write(!(x < 8 && x > 3)); // false     like "not"  in Python

2-3-1. While Loops:

Python code: <%# Copy source code %> 
num = 0
while num < 10:
    num += 1
    print(num) # print 1 ~ 10


# break
num = 0
while num < 10:
    num += 1
    if num == 5:
        break # print 1 ~ 4
    print(num)



# continue
num = 0
while num < 10:
    num += 1
    if num == 5: # skip 5
        continue
    print(num) # print 1 ~ 10
Javascript code: <%# Copy source code %>

while (num < 10) {
    num += 1;
    document.write(num); // print 1 ~ 10
}

// break
num = 0
while (num < 10) {
    num += 1;
    if (num == 5) {
        break;
    }
    document.write(num); 
}

// continue
num = 0
while (num < 10) {
    num += 1;
    if (num == 5) {
        continue;
    }
    document.write(num); // print 1 ~ 10, but skip 5
}

2-3-2. For Loops:

Python code: <%# Copy source code %> 
alphabet = ['a', 'b', 'c', 'd', 'e'] # a list

for i in alphabet:
    print(i) # a b c d e


for num in range(11):
    print(num) # print 0 ~ 10, default start is 0, increment is 1


for num in range(1, 11):
    print(num) # print 1 ~ 10, default start is 0, increment is 1


for num in range(1, 11, 2):
    print(num) # print odd number in 1 ~ 10 


for num in range(1, 11, 2):
    if num == 5:
        break;
    print(num) # print odd number in 1 ~ 5 



for num in range(1, 11, 2):
    if num == 5:
        continue;
    print(num) # print odd number in 1 ~ 10 but skip 5

2-4. Function:

Python code: <%# Copy source code %> 
def py_function_1(input):
    return input*input # create a function

print(py_function_1(3)) # call a function, 3*3=9

# anonymous function 
x = lambda input : input*input
print(x(3))



# global and local variable
global_var = 'global' # global variable
def py_function():
    local_var = 'local' # local variable
    print(global_var)
    print(local_var)

py_function()

# Nonlocal variable are used in nested function whose local scope is not defined.
global_var = 'global' # global variable
def outer_function():
    print(global_var)
    local_var = 'local'# local variable
    print('outer : ', local_var)
    def inner_function():
        nonlocal local_var # nonlocal variable
        # this variable is neither in the local nor the global scope
        local_var = "nonlocal"
        # in nested function. if we change "nonlocal" variable ,
        # the changes appears in the local variable.
        print('inner : ', local_var)
    inner_function()

outer_function()
